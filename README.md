# Heimdall Serivce
Simple scalable websocket gateway service

## Features
- Routes websocket request

## Getting Started
- Golang 1.11 required
- `go run cmd/heimdall`
- connect websocket to 0.0.0.0:8080/connect

## Error code
0 : no error
4001 : unauthorized request
4003 : forbidden request
4004 : unlisted request type
